This directory contains a collection of scripts which are intended for on-site asset discovery. All the scripts contained in this package can be run on a Windows 10 machine with powershell, bash on ubuntu on windows, and python.

DNSenum
    - A DNS enumeration tool

OSClassify
    - A ICMP based OS detection tool

Ping
    - A ping sweep tool

PortScan
    - A port scanning tool

SSHScan
    - A brute force SSH login scanning tool
